# web_href
脚本优势：
1.可快速获取功能点寻找可用的测试点。
2.支持批量获取，一键导出文件。
3.操作简单无任何含金量。
这里提供了windows和linux保存到桌面的两个方式如果需要保存到linux里面就把windows的注释，linux的取消注释即可
